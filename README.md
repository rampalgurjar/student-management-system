# Student Management System

A Python-based Student Management System built using Object-Oriented Programming (OOP) concepts and JSON file handling.

## Features

* Add Student
* View Students
* Search Student
* Update Student
* Delete Student
* Save Student Data in JSON File
* Load Student Data from JSON File

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* JSON
* File Handling

## Project Structure

```text
student-management-system/
│
├── main.py
├── student.py
├── studentmanager.py
└── data.json
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/rampalgurjar/student-management-system.git
```

2. Open the project folder

```bash
cd student-management-system
```

3. Run the project

```bash
python main.py
```
